About me

Hi! My name is Maxwell Griffin, I am entry level Software Developer who recently graduated from Eleven Fifty Academy located in Fishers, IN. I completed their twelve week immersive C# .NET boot camp which also incorporated HTML, CSS, and TypeScript, JavaScript. We used GitHub as our source code repository. Welcome to my portfolio, which uses Twitter Bootstrap for styling.

I developed two separate ASP.NET Web MVC applications. These applications utilized controllers, views, razor syntax, model classes, user authentication, and a SQL database with several tables with complete with CRUD support.

Another application I created was built using Angular/TypeScript on the front end and utilized Angular components and services. For the back end a C# Web API serving as a REST interface was used to create and retrieve data from SQL Server.

We used Visual Studio for the pure C#/.NET development and Visual Studio Code for the Angular TypeScript front end development.
