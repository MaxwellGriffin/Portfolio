Message Board

Message Board is a project that I created for my .NET Blue Badge at Eleven Fifty Academy. It is an ASP.NET Web MVC that I created in Visual Studio. Message Board is designed to emulate the old internet forums that I used to enjoy visiting. It uses a SQL database with 3 custom tables that I created complete with full CRUD on each. Message Board also has custom user options like the ability to change your profile picture. Some cool features include the ability to format text within a post using custom formatting tags, including the ability to post functioning links and images. This is accomplished through a text-parsing state machine.

Vacations United

Vacations United is my group project which was created to earn my .NET Red Badge at Eleven Fifty Academy. It uses an Angular front end and ASP.NET Web API back end and is styled using Angular Material. The purpose of Vacations United is to be able to create a vacation Group, an Itinerary, and select Destinations from a pre-made list, adding them to your Itinerary on specific days to outline your "vacation" across the United States.
